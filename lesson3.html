JSON Parse, Stringify

Lessons learned in the classs

JSON.parse()

		JSON.parse('{}'); // {}
		JSON.parse('true'); // true
		JSON.parse('"foo"'); // "foo"
		JSON.parse('[1, 5, "false"]'); // [1, 5, "false"]
		JSON.parse('null'); // null

JSON.stringify())
